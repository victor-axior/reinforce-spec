## Prerequisites

Before you begin, ensure you have:

| Requirement | Version | Notes |
|-------------|---------|-------|
| Python | 3.11+ | Required for SDK |
| OpenRouter API Key | — | [Get one here](https://openrouter.ai) |
| `pip` or `uv` | Latest | Package manager |
